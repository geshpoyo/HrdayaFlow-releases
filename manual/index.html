02 · CANVAS

無限キャンバスとジェスチャー

どこにでも置ける自由度と、揃えれば整然となるオートレイアウトのハイブリッド。

パン & ズーム Ctrl+ホイール
中ボタンドラッグでパン、Ctrl+ホイールでカーソル位置中心ズーム (5%〜800%)。
矩形・投げ縄選択 Shift+ドラッグ
完全包含/Crossing 切替。Shift+クリックで選択追加、Ctrl+A で全選択。
グリッド & マグネティック Ctrl+;
8/16/32/64 px グリッド、エッジ吸着、ガイド線スナップ。Alt で一時無効。
整列・配置
左/中央/右、上/中央/下揃え。横/縦方向の均等配置。複数選択時に右クリック。
Blender 風変形 G/S/R
Grab/Scale/Rotate モード。X/Y で軸固定、数値タイプで量指定。
重ね順 Ctrl+]/Ctrl+[
1 段ずつ前後、Ctrl+Shift で最前/最背。
ハンドルドラッグでエッジ作成
ノート端のハンドルから他ノートへ。Shift で矢印、Alt で逆向き、Ctrl で双方向。
グループ化 Ctrl+G
複数選択 → Container 化、折りたたみ可。Ctrl+Alt+G で内部 Auto Layout。
サブマップ昇格
Container を別タブで開いて編集。複雑な部分を分離整理。
レイヤー
表示/非表示・ロック・色付け。エッジだけのレイヤーも作れる。

03 · FORMAT

Inspector & 書式パネル

Scapple の NOTE FORMAT 相当 + 拡張。複数選択で一括変更も。

B / I / U / S Ctrl+B
太字・斜体・下線・取り消し線。範囲選択で部分適用 (InlineMark)。
フォントサイズ
8〜144px のスライダー or 数値入力。Ctrl++/Ctrl+- で 2px ステップ。
配置 Ctrl+L/E/R
水平 (左/中/右) / 垂直 (上/中/下) を独立に。
カラーピッカー
プリセット 16 + カスタム HEX + Recent 8。テキスト色 / 塗り / 枠 / 影に独立指定。
図形 (8 種)
矩形/角丸/楕円/ひし形/六角/雲/平行四辺形/三角。Ctrl+Shift+1〜8 で切替。
Border
幅 0〜10px、Style (実線/破線/点線/二重)、色。Ctrl+Shift+B でトグル。
Drop Shadow
ON/OFF 切替。色・距離はテーマ連動。
Opacity
0〜100% スライダー。0% でも選択は可。
Magnetic Snap
他ノートのエッジと整合する位置で吸着。重要ノートだけ ON 推奨。
種別切替 (Kind Switcher)
text → markdown → image など、データを保ったまま変換。

05 · AI

AI 連携 (3 backend)

stub (オフライン) / Ollama (ローカル LLM) / PrudeLLM (ホスト)。テキスト⇔マップ往復、エッジ提案、子提案。

テキスト → マップ Ctrl+Enter
箇条書きや文章をノート + エッジに構造化。WebSocket でストリーミング表示。
マップ → 要約
箇条書き / 段落 / Twitter 280字 / 1行 から文体選択。クリップボード or 新規 Markdown へ。
Suggest Edges
繋がっていない関連ノート間のエッジを提案。信頼度スコア付き。
Suggest Children Ctrl+Shift+R
1 ノートに対して子候補を生成。「目的・期日・担当・リスク・予算」など。
Diff (JSON Patch)
2 マップの差分を RFC 6902 で取得。バージョン管理・共同編集の基礎。
MCP Server (HTTP)
Claude Desktop / Cursor などから操作可能。read / find_node / bulk_apply / bulk_patch。
Ollama (ローカル)
localhost:11434 を自動検出。プライバシー重視のユーザー向け。
PrudeLLM (ホスト)
API キーで利用。明示同意制。マップ内容が外部送信される。
Stub フォールバック
LLM タイムアウト時に rule-based に自動切替。アプリは止まらない。

06 · STACK

3 層アーキテクチャ

UI = Flutter で機敏に、Core = Rust で安全・高速に、AI = Python で隔離。

UI

Flutter Desktop (Dart 3.x)

キャンバス描画 (CustomPaint) / ジェスチャー / ノード編集 / パネル / Inspector / Undo Adapter

↑ ↓ FFI (flutter_rust_bridge 2.12)
CORE

Rust (hrdaya_core)

graph / layout / .hrf v0.2 ZIP I/O / importers (7 形式) / exporters (PDF/SVG/PNG/Md) / JSON Patch (RFC 6902)

↑ ↓ HTTP / WebSocket :8765
AI

Python AI Service (FastAPI)

/api/generate-map / /api/summarize-map / Backend: stub / PrudeLLM / Ollama
