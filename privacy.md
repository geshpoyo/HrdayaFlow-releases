# プライバシー方針（自動更新）

HrdayaFlow の自動更新チェックに関する送受信情報は以下です。

## 送信する情報

- アプリバージョン（`User-Agent`）
- 更新チャンネル（`stable` / `beta`）
- プラットフォーム情報（例: `windows-x64`）

User-Agent 例:

`HrdayaFlow/0.0.3 (windows; STABLE)`

## 送信しない情報

- マシン固有 ID
- 利用統計・操作ログ
- マップ本文や添付データ
- ユーザーアカウント情報

## 停止方法

- `設定 > 更新` の「自動的に更新を確認」を OFF にすると、自動チェックは停止します。
- 手動チェック（`今すぐ確認`）はユーザー操作時のみ行われます。

## 備考

- サーバー側の通常アクセスログ（IP/時刻）はホスティング基盤の標準挙動として記録される場合があります。
